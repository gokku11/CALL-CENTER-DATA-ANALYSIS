# Call Center Dashboard

## Project Overview
This project provides a simple Call Center Dashboard built in Microsoft Excel to analyze customer service performance and call center operations.

## Features
- Total Calls Analysis
- Customer Satisfaction (CSAT) Tracking
- Call Duration Monitoring
- Sentiment Analysis
- Call Channel Distribution
- State and City-wise Insights
- Interactive Filters and Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Visualization

## Objectives
- Monitor call center performance
- Analyze customer satisfaction trends
- Identify common customer concerns
- Support data-driven decision making
